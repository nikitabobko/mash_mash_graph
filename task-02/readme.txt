Реализована вся базовая часть скринсейвера:
* Используется 3 различных типа фигур (куб, тетраэдр, треугольник)
* 6 граней цветного куба раскрашены в 6 различных цветов
* На экране рисуется 10 фигур
* Все фигуры во время движения вращаются вокруг своего геометрического центра (в случайные 
  моменты меняют направление своего вращения)
* Все движения фигур - плавные
* Только 2 из 10 фигур плоские, все остальные 3D
* Скорость работы программы не зависит от частоты обновления экрана. То есть
  фигуры не двигаются быстрее на более быстрой машине.

За базовую часть, как и полагается, претендую на 10 баллов.

Доп. часть:
* Использование фона с текстурой (+1)
* Использование мешей, загруженных из файла (Мьёльнир загружен из файла,
  расположенного по пути res/obj/thor_hammer.obj) (+3)
* Использование текстур на фигурах (+1)
* Возможность разместить камеру на случайном объекте (по нажатию F2 камера фиксируется 
  на случайном объекте, при нажатии на F1 возвращается в изначальное положение) (+1 балл)

Итого за доп. часть: 6 баллов.

Установка зависимостей:
sudo apt update
sudo apt install g++
sudo apt install cmake
sudo apt install libglfw3-dev
sudo apt install libassimp-dev
sudo apt install libglm-dev
sudo apt install libsoil-dev

Как собирать и запускать:
mkdir build
cd build
cmake ..
make
./main
